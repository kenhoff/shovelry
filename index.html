For Shovelry!

Shovelry is a tiny library to add Shovel Knight-themed strong and emphasis text to any page.

Check out the repo on GitHub to download and contribute!

Usage

<script src="shovelry.js" charset="utf-8"></script>
<link rel="stylesheet" href="shovelry.css">

<p>
    Why is it called the <em>Dark Ages</em>?
    Because there's no shortage of <strong>knights!</strong>"
</p>

<script type="text/javascript">
    shovelry();
</script>

Options

shovelry({
    // haha, fooled you! there are no options.
});

FAQ

Does it work with non-monospaced text?

Sure does! Serifed, sans-serifed, fantasy, cursive, even Comic Sans!

Why does mine still have the bold/italics on the text?

I've pulled off the font-style and font-weight styles on this page to make it a little more Shovel Knight-ly. You can do it yourself by just setting some CSS rules: 

em {
    font-style: normal;
}
strong {
    font-weight: normal;
}
